## ⚠️ قوانین مهم

### 1. ساختار داینامیک:
- **گزارش می‌تواند شامل چندین دسته‌بندی/محصول باشد** - برای هر کدام یک تب/صفحه بساز
- **تعداد سوالات متغیر است** - هر سوالی که کاربر داد را اضافه کن
- **نوع نمودار را از داده‌ها تشخیص بده:**
  - اگر داده‌ها درصدی هستند → نمودار دونات یا میله‌ای
  - اگر داده‌ها رتبه‌بندی هستند → نمودار میله‌ای افقی
  - اگر داده‌ها مقایسه‌ای هستند → نمودار میله‌ای گروهی

### 2. پردازش انعطاف‌پذیر جداول:
- **جداول می‌توانند هر ساختاری داشته باشند** - شما باید ساختار را از header استخراج کنید
- **نام ستون‌ها را تغییر نده** - همانطور که کاربر داده نمایش بده
- **اگر ستون جدیدی در جدول وجود دارد** → آن را هم اضافه کن
- **اگر ستونی وجود ندارد** → محاسبه نکن یا از ستون مناسب دیگر استفاده کن

### 3. نمودارها:
- **از Chart.js استفاده کن** - کتابخانه موجود در فایل پایه
- **نمودارها باید تعاملی باشند** - با hover و tooltip
- **برچسب‌های فارسی را درست نمایش بده**
- **رنگ‌بندی یکپارچه و حرفه‌ای استفاده کن**

### 4. صفحات و تب‌ها:
- **اگر چند دسته‌بندی داریم** → تب‌های قابل کلیک بساز
- **اگر یک دسته‌بندی داریم** → تب نیاز نیست
- **صفحه Landing همیشه اول** - با لوگوها و عنوان گزارش
- **صفحه Insights همیشه آخر** - با برداشت‌های کلیدی

### 5. پاک کردن داده‌های قبلی (خیلی مهم!):
**⚠️ هرگز داده‌های برندهای قبلی را نگه ندار!**

- **قبل از شروع:** تمام داده‌های قبلی را از فایل پایه پاک کن:
  - جداول را خالی کن یا با داده‌های جدید جایگزین کن
  - نمودارها را با داده‌های جدید به‌روزرسانی کن
  - کارت‌های آماری را با داده‌های جدید به‌روزرسانی کن
  - نام برند را در همه جا به‌روزرسانی کن
  - لوگو را به‌روزرسانی کن

- **بررسی نهایی:** مطمئن شو که هیچ داده‌ای از برندهای قبلی باقی نمانده:
  - در جداول
  - در نمودارها
  - در کارت‌های آماری
  - در متن‌ها و توضیحات

### 6. فرمت داده‌ها:
- **اگر داده JSON است** → مستقیماً استفاده کن
- **اگر داده جدول است** → پارس کن و به JSON تبدیل کن
- **اگر داده Excel است** → از کاربر بخواه که به CSV یا JSON تبدیل کند

## 📊 مرحله 6: ساخت نمودارها با Chart.js

### 6.1 نمودار دونات (برای داده‌های درصدی):

```javascript
new Chart(ctx, {
  type: 'doughnut',
  data: {
    labels: [گزینه‌ها],
    datasets: [{
      data: [درصدها],
      backgroundColor: [رنگ‌ها]
    }]
  },
  options: {
    plugins: {
      datalabels: {
        formatter: (value) => value + '%'
      }
    }
  }
});
```

### 6.2 نمودار میله‌ای افقی (برای رتبه‌بندی):

```javascript
new Chart(ctx, {
  type: 'bar',
  data: {
    labels: [گزینه‌ها],
    datasets: [{
      data: [مقادیر],
      backgroundColor: [رنگ‌ها]
    }]
  },
  options: {
    indexAxis: 'y',
    plugins: {
      datalabels: {
        formatter: (value) => value + '%'
      }
    }
  }
});
```
