# bb-150 — 400 × 150 Yektanet Digital Billboard

## Frame

```
Width:    400px   Height: 150px   Direction: RTL   Mobile-only
Body bg:  transparent

Design zone:  bottom 120px (y=30 → y=150)  ← panel and text live here
Poke-out zone: top 30px  (y=0  → y=30)   ← hero characters and products overflow here
```

```css
* { margin: 0; padding: 0; box-sizing: border-box; }
body {
  width: 100%; height: 150px;
  overflow: hidden; background: transparent;
  font-family: 'Vazirmatn','IRANYekan','YekanBakh','Tahoma',sans-serif;
}
```

**BG panel typical anchor:**
`position: absolute; top: 30px; right: 5px; width: 245px; height: 115px; border-radius: 14px;`

This covers ~65% of the 400px frame width → satisfies 70/30 rule.
Hero floats as a sibling (not inside panel), overlapping into the poke-out zone.

## Tags — postMessage

```javascript
const ALL_EVENT_TYPES={WINDOW_LOADED:'WINDOW_LOADED',DOM_CONTENT_LOADED:'DOM_CONTENT_LOADED',TIMER_0_SECOND:'TIMER_0_SECOND',TIMER_5_SECOND:'TIMER_5_SECOND',TIMER_10_SECOND:'TIMER_10_SECOND',TIMER_15_SECOND:'TIMER_15_SECOND',TIMER_60_SECOND:'TIMER_60_SECOND',VISIT_SLIDE01:'VISIT_SLIDE01',VISIT_SLIDE02:'VISIT_SLIDE02',VISIT_SLIDE03:'VISIT_SLIDE03',VISIT_SLIDE04:'VISIT_SLIDE04',VISIT_SLIDE05:'VISIT_SLIDE05',VISIT_SLIDE06:'VISIT_SLIDE06',CLICK1:'CLICK1',CLICK2:'CLICK2',CLICK3:'CLICK3',CLICK4:'CLICK4',CLICK5:'CLICK5',CLICK6:'CLICK6',CLICK_AUTOPLAY:'AUTOPLAY1',LOOP:'LOOP'};
function fire_tag(t){if(!Object.values(ALL_EVENT_TYPES).includes(t)){console.warn('TAG NOT ALLOWED:',t);return;}window.parent.postMessage({type:'yn::event',event_type:t},'*');}
fire_tag(ALL_EVENT_TYPES.TIMER_0_SECOND);
window.onload=()=>fire_tag(ALL_EVENT_TYPES.WINDOW_LOADED);
document.addEventListener('DOMContentLoaded',()=>{fire_tag(ALL_EVENT_TYPES.DOM_CONTENT_LOADED);let t=Date.now(),f5=true,f10=true,f15=true,f60=true;let iv=setInterval(()=>{const s=(Date.now()-t)/1000;if(s>=5&&f5){f5=false;fire_tag(ALL_EVENT_TYPES.TIMER_5_SECOND);}if(s>=10&&f10){f10=false;fire_tag(ALL_EVENT_TYPES.TIMER_10_SECOND);}if(s>=15&&f15){f15=false;fire_tag(ALL_EVENT_TYPES.TIMER_15_SECOND);}if(s>=60&&f60){f60=false;fire_tag(ALL_EVENT_TYPES.TIMER_60_SECOND);clearInterval(iv);}},1001);});
```

**Tracking pattern:**
- `VISIT_SLIDE01` — on `DOMContentLoaded`
- `VISIT_SLIDE02`–`VISIT_SLIDE06` — at start of each scene via `tl.call`
- `CLICK1`–`CLICK6` — per clickable element (`addEventListener('click', ...)`)
- `LOOP` — in `tl.onComplete` before reload

## Layout Patterns

**Pattern A — 70/30 Split (default, most versatile):**
```
[LEFT ~40%: hero poke-out]   [RIGHT ~60%: dark panel]
 • character / product         • logo badge (top-right)
   top-anchored, bleeds up      • headline (mid)
   into poke-out zone           • subheadline (below)
                                • CTA pill (bottom)
```

**Pattern B — KV Flood then Panel Reveal:**
- S1: KV fills full frame (0% bg, KV exception applies). Viewer's eye caught.
- S2: Panel slides in from right. Text builds on top. Coverage jumps to ~60%.
- S3: CTA finale. Panel stays, hero shrinks to side. Coverage ~55%.

**Pattern C — 3-Act Narrative:**
- Act 1 (0–5s): Brand logo only, minimal bg (~20%)
- Act 2 (5–18s): Hero + value prop, bg rises to ~55%
- Act 3 (18–28s): Urgency + CTA + tapesh pulse, bg holds ~50%

Default to A. Use B when the hero is large and striking. Use C for brand-awareness builds without a dominant KV.

## Animation Budget

- **Scenes:** 4–6 distinct scenes (not 2, not 7)
- **Runtime:** 22–34s — narrative-driven, never padded with empty time
- **First brand element visible:** ≤500ms from load (no slow intros)
- **Loop:** `tl.onComplete` → kill infinite tweens → fire LOOP → `setTimeout(() => location.reload(), 100)`
- **No viewport trigger:** bb-150 is sticky-bottom, always visible. Timeline starts on load.

---

## Format-Specific Constraints

- Body is `transparent` — the Yektanet sticky slot floats over the publisher page
- Hero elements must use `position: absolute` (not fixed) — they move within the 400×150 frame
- BG panel `border-radius: 14px` is the standard — gives the floating-card look
- Avoid placing text below y=140 (last 10px is a dead zone in many placements)
- On mirror/flip scenes: animate with `scaleX: -1` on the hero wrapper, not by changing `right`/`left` values
